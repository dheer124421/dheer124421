### 🚀 What I'm building

I'm currently focused on the layer where LLMs meet automation — turning a model's output into a real, working system rather than just a chat response. Below are my 3 main AI projects:

**🔹 [AI Email Reply Generator](https://github.com/dheer124421/AI-Email-Reply-Generator)**
End-to-end n8n workflow that reads unread Gmail messages, drafts context-aware replies using Groq LLM, logs everything in Airtable, and marks emails as read — zero manual intervention.
`n8n` `Groq LLM` `Gmail API` `OAuth 2.0` `Airtable`

**🔹 [Video RAG Chatbot](https://github.com/dheer124421/Video_RAG_Chatbot)**
A Retrieval-Augmented Generation chatbot that answers questions about a 2-hour YouTube interview, returning clickable source timestamps for verification. Built with semantic search (cosine similarity) over transcript embeddings.
`Python` `Flask` `Sentence Transformers` `Gemini 2.5 Flash`

**🔹 [AI Telegram Task Bot](https://github.com/dheer124421/telegram-ai-task-bot)**
Telegram bot that converts text or voice messages into structured ClickUp tasks using Gemini for extraction and Groq Whisper for speech-to-text — no manual task entry needed.
`n8n` `Telegram Bot API` `Groq Whisper` `ClickUp API`
